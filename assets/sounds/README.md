# Sound Effects

Drop free MP3s here with these exact filenames. Each is optional — if
missing, the app silently falls back to haptic feedback only.

| File | When it plays |
|---|---|
| `coin.mp3` | Coin drop (Add Coin, task approved) |
| `success.mp3` | Level up / milestone / mission complete |
| `pop.mp3` | Button taps (optional) |
| `oink.mp3` | Tapping the pig (optional) |

Free CC0 sources:
- https://freesound.org (search: "coin", "success chime", "pop", "pig oink")
- https://pixabay.com/sound-effects/
- https://mixkit.co/free-sound-effects/game/

Keep clips under 1 second for snappy UI feel.
